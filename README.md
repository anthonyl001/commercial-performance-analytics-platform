## Overview

Developed an enterprise-level Commercial Performance Analytics Platform for Pernod Ricard Thailand to provide end-to-end visibility into sales performance, distribution effectiveness, and commercial execution.

The solution was built using Microsoft Power BI and Azure-based data infrastructure, enabling both executive management and operational teams to access business-critical insights through a centralized reporting platform.

# Executive Dashboard

The Executive Dashboard was designed for senior management and commercial leaders to monitor overall business performance and strategic KPIs.

### Key Features

- Sell In Monitoring
- Sell Out Monitoring
- Sales Growth Tracking
- Target Achievement Analysis
- Regional Performance Analysis
- Wholesaler Performance Ranking
- Brand Performance Monitoring

### Dashboard Preview

![Executive Dashboard](images/executive-dashboard.png)

### Key Metrics

| Metric | Description |
|----------|-------------|
| Total Sell In | Total value sold to distributors |
| Total Sell Out | Total value sold to end customers |
| Growth % | Month-over-month or year-over-year growth |
| Distance to Target | Remaining gap to sales target |
| Active Outlet | Number of active selling outlets |
| Active Wholesaler | Number of active wholesalers |
| Distribution % | Product distribution coverage |

### Business Value

- Provides a consolidated view of commercial performance.
- Enables management to monitor achievement against sales targets.
- Identifies top-performing wholesalers and regions.
- Supports strategic planning and resource allocation.

---

# Self-Service Dashboard

The Self-Service Dashboard was designed for operational users including Sales Managers, Trade Marketing Teams, and Commercial Analysts.

### Key Features

- Brand Performance Analysis
- Product Performance Analysis
- Wholesaler Analysis
- Point of Distribution Monitoring
- Distribution Coverage Analysis
- Channel Performance Monitoring

### Dashboard Preview

![Self Service Dashboard](images/self-service-dashboard.png)

### Key Metrics

| Metric | Description |
|----------|-------------|
| Sell In by Brand | Sales contribution by brand |
| Sell Out by Brand | Consumer sales by brand |
| Point of Distribution | Outlet distribution status |
| Point Coverage | Product availability coverage |
| Channel Performance | Sales contribution by channel |
| Wholesaler Performance | Sales performance by wholesaler |

### Business Value

- Allows users to perform detailed operational analysis.
- Reduces dependency on manual Excel reporting.
- Enables faster identification of distribution gaps.
- Supports sales execution monitoring.
